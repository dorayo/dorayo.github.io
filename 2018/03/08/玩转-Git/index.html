00. Git 操作一览


术语翻译：

  • Workspace：工作区
  • Index / Stage：暂存区
  • Repository：版本库、仓库（本地仓库）
  • Remote：远程仓库

补充说明：

上图不是太准确，在Remote 和 Repository 内部都还有个 Branch 的概念，单不太影响整体理解。
